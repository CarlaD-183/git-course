 1. Giải thích các câu lệnh: git init, git add <file_name>, git add ., git commit -m “message”.
- Git init: khơỉ tạo thư mục được quản lý bởi git
- Git add <filen_name> thêm 1 file mới vào vùng staging 
- Git add . : thêm tất cả các file từ local vào staging
- Git commit -m"mesage" : đưa file từ vùng staging lên repository vùng với 1 ghi chú cho việc thêm file / feature này "mesage"
